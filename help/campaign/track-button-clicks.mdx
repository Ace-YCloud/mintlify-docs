# Track Button Clicks

> **Info**
>
> Companies can enable button tracking for the 'Visit Website' button when applying for utility and marketing templates. This feature allows companies to view click data for the button on the corresponding analysis page after completing the campaign. It helps companies with internal data tracking and analysis.

## Step 1

Create template > Enable Button tracking
