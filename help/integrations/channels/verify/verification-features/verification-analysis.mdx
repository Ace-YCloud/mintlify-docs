# Verification Analysis

YCloud verify offers a complete data cycle from verification request to sending to success, automatically sending verification reports to businesses daily. Each verification log is clear and traceable.

> **Info**
>
> 🚧 Note: If you are using [Custom OTP](/help/integrations/channels/verify/verification-features/custom-otp), we may not be able to show you the success rate of verifications. Therefore, we recommend that regardless of whether you use custom OTPs, please provide feedback through the verification API to let us know if the user has been verified. This can also help us monitor global communication channels to ensure the highest message delivery rate.
