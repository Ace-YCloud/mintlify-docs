# Verification Message Templates

This template is predefined and approved text used for sending verification messages, supporting channels including SMS, voice, WhatsApp, and email.

If you have other business needs, please contact us.

## Entry

Integrations > Verify > Settings, [Click to jump to the settings page](https://www.ycloud.com/console/#/app/integrations/verify/setting)

![](/images/gitbook/image-488-fc4cbc3c.png)

## Customizable Message Template

The message template supports customization of the following content, which can be configured in the console:

| Customization Option                  | Definition                                                                                                                                                                                                                                               | Supported Channels                              |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| Brand                                 | The brand name and logo displayed in the verification message                                                                                                                                                                                            | All channels except voice.                      |
| Verification Code Length              | The longer the verification code in the message, the higher the security level.                                                                                                                                                                          | All channels.                                   |
| Verification Code Validity            | Once expired, the verification code becomes invalid and cannot be used for verification.                                                                                                                                                                 | All channels.                                   |
| "Verification Code Validity" Reminder | If enabled, the validity information will appear at the end of the message.                                                                                                                                                                              | All channels except voice.                      |
| "Do Not Share" Alert                  | If enabled, "Do not share this code with anyone." will appear at the end of the message.                                                                                                                                                                 | All channels except voice and WhatsApp.         |
| "SMS Fallback" Alert                  | If enabled, when the WhatsApp status report returns "failed," YCloud will automatically attempt to resend the verification code via SMS. However, if the status report is delayed by more than 30 seconds, YCloud will not automatically resend via SMS. | Only WhatsApp supports customizing this option. |

## Supported Languages

We will automatically parse the country/region code of the provided phone number into the local language, with English as the fallback language. If you still need to specify a language, you can override it using the language parameter.

Supported languages include:

ar: Arabic\
de: German\
en: English\
es: Spanish\
fr: French\
id: Indonesian\
it: Italian\
pt: Portuguese\
ru: Russian\
tr: Turkish\
vi: Vietnamese\
zh: Simplified Chinese\
zh-hk: Traditional Chinese.

> **Info**
>
> Note that this does not apply when the channel is voice.
