# Security Settings

We have provided detailed information on preventing fraud and invalid registrations in our articles. If you realize you are under attack or wish to block such attempts, you can use YCloud's preset security settings, including checking the phone number before initiating verification, restricting countries/regions, and blacklisting restrictions. Additionally, we will send you a daily verification report via email to help you monitor and mitigate potential losses. However, no provider's solution can guarantee 100% effectiveness against experienced attackers, so your involvement is crucial.

You can set up the following security measures in the YCloud console:

* **Invalid Number Filtering**: Most landline phones cannot receive verification messages, so you can choose to prohibit sending messages to landline numbers to avoid wasting resources.
* **Geographical Permissions Restriction**: Select countries/regions where you do not conduct business, and messages sent to these countries/regions will be blocked. This is an effective preventive measure. (Not applicable to emails)
* **Blacklist**: Add explicitly blacklisted phone numbers, email addresses, or email domains. Verification messages will no longer be sent to them.

## Setup Guide

Integrations > Verify > Security, [Click to Access](https://www.ycloud.com/console/#/app/integrations/verify/security)
