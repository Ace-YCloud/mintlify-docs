# Order canceled/fulfilled

When the order status is changed to cancel/fulfilled, automatically remind customers through WhatsApp.

These two types of message settings are the same. This article takes the order canceled as an example to introduce.


## Step1: Visit [Shopify Journey](https://www.ycloud.com/console/#/app/integrations/shopify/shopifyJourney)

![](/images/gitbook/image-625-b7df32e4.png)

## Step2: Set the details

**Sender**: which WhatsApp business phone number to send message.

**Send template**: choose a template

![](/images/gitbook/image-626-17c560bb.png)

## Step3: Save and active the Journey

![](/images/gitbook/image-627-d85633b3.png)
