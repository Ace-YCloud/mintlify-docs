# MM Lite API

Marketing Messages Lite API ("MM lite API") is a new way to send WhatsApp marketing messages, which offers businesses the ability to leverage Meta’s AI system to drive business performance and improve quality.

## Who is it for?

Marketers with limited ad campaign expertise and do not require extensive features of Marketing Messages API today

## What do you get?

✅ Higher or neutral delivery rates compared to Cloud API (Which you are using now)

✅ Track when marketing messages lead to actions on a business’ website (e.g. add to cart, checkout, purchase)

✅  Add a valid period time for the Marketing template.

## How can I use it?

Authorize your WABA to send messages via the [MM Lite API](https://www.ycloud.com/console/#/app/integrations/mmLiteApi) and all your marketing messages to regions covered by the MM Lite API will be automatically delivered through it.

Click the **+Authorize button** and finish the authorization.

## Geographic availability of features <a href="#geographic-availability-of-features" id="geographic-availability-of-features"></a>

Some advanced features and reporting capabilities of MM Lite API are available only in particular geographies due to Meta policy and/or local regulation.

**United States:**

* Starting Apr 1, 2025, marketing messages sent to WhatsApp users in the United States will not be delivered (error code 131049). Note that this policy is not specific to MM Lite API - it is in place across all Business Messaging APIs (including Cloud API, [see docs](https://developers.facebook.com/docs/whatsapp/cloud-api/guides/send-message-templates#per-user-marketing-template-message-limits)).
* Businesses phone numbers in the US will continue to be able to use MM Lite API to message users outside of the United States.

**Cuba, Iran, North Korea, Syria, and three sanctioned regions in the Ukraine (Crimea, Donetsk, Luhansk):**

* Businesses in these regions are not eligible to onboard, and messages cannot be sent to a consumer in these regions. This policy is not specific to MM Lite API - it is in place across all Business Messaging APIs (including Cloud API, [see docs](https://developers.facebook.com/docs/whatsapp/cloud-api/guides/send-message-templates#per-user-marketing-template-message-limits)).
