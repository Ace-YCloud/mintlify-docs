# Create WhatsApp Flows Template

> **Info**
>
> YCloud currently does not support creating WhatsApp Flows in the YCloud backend. You will need to create them in your own BM (Business Manager) backend. However, please note that only accounts with business verification can create Flows.

This document provides both video and text-based tutorials for creating WhatsApp Flows. If you would like to view the video tutorial, please click here to watch. Next is the detailed step-by-step tutorial with images.

## Step 1:

Log in to your BM account > Flows > Create flow

![](/images/gitbook/image-178-5e28b8ea.png)

## Step 2:

Enter the template name, select the template category, and choose the display format of the template.

Note: In the bottom-right corner, you can select different template formats.

![](/images/gitbook/image-179-f0f4b6cc.png)

## Step 3:

Click "Publish" to publish this template.<br />

![](/images/gitbook/image-180-8257c6e5.png)

## Step 4:

When the below page appears, it means the Flow has been successfully created!

![](/images/gitbook/image-181-79dc809f.png)

## Step 5:

> **Info**
>
> Only marketing/utility templates support the insertion of WhatsApp Flows.

Log in to your YCloud account > Templates > Select the appropriate template type and edit the content > Click "Buttons" > "Call to action" > "WhatsApp Flow" > Select the WhatsApp Flow you want to insert > Edit the text displayed on the button > Click "Submit" to submit the template for approval.

![](/images/gitbook/image-590-6e453a42.png)
