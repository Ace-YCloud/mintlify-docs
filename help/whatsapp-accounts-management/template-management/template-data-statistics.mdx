# Template Data Statistics

## Data Statistics Scope

1. Delivered: Number of messages arrived.
2. Read: Number of messages read. This Status could not be obtained when users close terminal read receipts.&#x20;
3. Quick reply button clicks: Number of clicks on the Quick reply type buttons in the template.
4. Visit website: Not counted by default. This feature requires contacting YCloud customer service.

Data statistics are defaulted to GMT+8.

## Step 1: Click on the Template

![](/images/gitbook/image-200-1-1f99a377.png)

*Click template*

## Step 2: View Template Data

1. Performance: Data statistics on message delivery and read status.
2. Button clicks: Data on button clicks in the message. This section will not be displayed if the template does not contain any statistically trackable buttons.
