# Copy Template

To create templates quickly and easily, you can use the copy template feature. Recommended usage scenarios:

1. When there are multiple WABAs in the YCloud account, you need to copy the same template to multiple WABAs.
2. Within the same WABA, you need to create a template that is not identical but similar in content.
3. When a template fails the review due to an incorrect Category selection, you can use the copy template button to create a new template and select the correct Category.

## Step 1: Click the Copy Button

![](/images/gitbook/image-204-1-d2e81038.png)

*Copy template button*

## Step 2: Set Copy Information

1. Destination WhatsApp accounts: Target WABA. Select the WABA to which you want to copy the template, or you can choose the current WABA.
2. Template name: New template name. Name the new template.
3. Category: Template type. It is not recommended to adjust if the template has already been approved.

After setting the content, click Submit.

![](/images/gitbook/image-205-1-31be78ef.png)

*Set copy info*

## Step 3: Edit the New Template

Enter the new template's editing page, and after confirming the content is correct, click Submit.

![](/images/gitbook/image-207-1-aac1a771.png)

*Edit template*
