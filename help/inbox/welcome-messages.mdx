# Welcome Messages

When you want to reply with a pre-configured message regardless of the content of the inbound message when a customer enters a conversation, the welcome message feature can meet your needs.

![](/images/gitbook/My_Photor_1761294340779-1-89a108e7.jpg)

#### 1. Customer Service Agents Welcome Message

1. After entering Inbox, click the \[Settings] button on the right side of the avatar

![](/images/gitbook/My_Photor_1761293504820-5109967b.jpg)

2. Select the \[Welcome Message] menu to manage your welcome messages.

![](/images/gitbook/My_Photor_1761293592496-185f261f.jpg)

3. Click the "Add Welcome Message" button to create a new welcome message, select for which business numbers the welcome message will take effect, and support configuring three different types of welcome messages (text, media, interactive messages)

![](/images/gitbook/My_Photor_1761293665676-1-9a15c3dd.jpg)

![](/images/gitbook/My_Photor_1761293687669-65cc3be9.jpg)

4. You can make the welcome message officially effective by turning on the switch, and you can also edit or delete the welcome message by clicking the button in the upper right corner

![](/images/gitbook/My_Photor_1761293731204-e8dec5ae.jpg)

#### 2. Chatbot Welcome Message

You can also configure a welcome message for the Chatbot. When the number assignment rule prioritizes assignment to the Bot, it will automatically reply with the configured welcome message when the user sends an upstream message. This can be set via the "Welcome message sent at the start of a conversation" in Chatbot-Profile

![](/images/gitbook/My_Photor_1761294250010-20dd05c7.jpg)
