# Message Translation

### Translation settings

1. Click on the profile photo in the lower left corner, then click the "gear" button on the right side of the profile photo to enter the personal settings menu.
2. Click on "Translation" in the left menu.
3. The "Input Translation" switch controls the translation function in the Inbox input box. After turning it on, the commonly used language of the agent can be translated into the language used by the customer in the Inbox input box.
4. The "Automated Customer Message Translation" switch controls the automatic translation function of customer messages in the current conversation page of Inbox. After turning it on, the messages sent by customers can be translated into the selected target language in real time. The target language is the same as the system workbench language by default, and manual adjustment is allowed.
5. The number of characters used by the current tenant translation is displayed above the switch.

![](/images/gitbook/image-155-8cdbaa0c.png)

### Chat history translation

1. Go to the Inbox message dialog box, hover the message card you want to translate, and click the three dot buttons on the right.
2. Click the "Translate" button in the pop-up window that appears.
3. Select the desired language in the target language that appears for translation, and click to complete the translation of the message content.

![](/images/gitbook/image-156-da23ef7e.png)

### Automatic translation

1. After turning on the "Input Translation" switch on the personal settings page, enter the dialog box, and the customer messages on the current screen will be automatically translated into the target language.

### Input box translation

1. After opening "Input Translation" on the personal settings page, the corresponding translation preview interface will appear in the Inbox text box.
2. Click on the button \[1] to select the target language for translation (a language that the customer can understand).
3. Enter the message you want to send in the input box, click the translation button in the button \[2], and check the translation results that appear in the translation preview box. You can adjust it manually.
4. Click the Send Translation button in button \[3] to send the corresponding translation.

![](/images/gitbook/image-157-b90db082.png)

### Number of translated characters

The number of translated characters of the current tenant is showed in the traslation settings. You will get free characters according to your business plan. If there is a need for more characters, please buy add-ons for translation.
