# Inbox Data Analysis

## Data Analysis Entry

The entry is located at the bottom left of Inbox > [Analytics](https://www.ycloud.com/console/#/app/inbox/analytics)

![](/images/gitbook/image-393-3f6816c7.png)

## Real-time Overview

The Inbox Real-time Overview provides real-time information (updated every 1h). With this feature, you can clearly understand the number of conversations today, know the overall scale of communication with users today; grasp the real-time reception status of customer service. The statistical time zone is GMT+8, starting from 0:00. By default, it counts the situation under all WhatsApp accounts, and you can filter at the top to view the data of a specific WhatsApp number corresponding to the Inbox separately.

* **Today's Conversations:** Number of conversations entered every hour today

![](/images/gitbook/image-394-0cb823f0.png)

* **Open Conversations:** Includes opened conversations, unassigned conversations, and conversations that have been assigned but not yet responded to

![](/images/gitbook/image-395-54d106aa.png)

* **Agent Status:** Available or Away status for agent

![](/images/gitbook/image-396-39ad5f1c.png)

* **Agent Conversation Handling:** From the perspective of agents, understand the corresponding opened and unanswered conversations. Used to understand the performance and workload of agents.

![](/images/gitbook/image-397-b433d7bb.png)

## Agents

From the perspective of agents, the entry is shown in the figure Analytics > [Agents](https://www.ycloud.com/console/#/app/inbox/analytics)

* **Data Filtering:** Select the statistical time period, defaulting to the last 7 days. Up to the last 1 year of data can be selected. Select the corresponding agent, defaulting to the total data of all agents, click the "magnifying glass" search button to load the data.
* **Data Download:** Click the "`Download`" button

![](/images/gitbook/image-398-f4c8ee26.png)

#### Conversation Statistics Fields

Conversation statistics unit:

* day: d
* hour: h
* minute: m

<table><thead><tr><th width="372">Field</th><th>Description</th></tr></thead><tbody><tr><td>Conversations (Total)</td><td>The number of conversations received by the agent. Based on the creation time of the conversation, the time period it belongs to is counted.</td></tr><tr><td>Online time (Total)</td><td>The length of time the agent is online.</td></tr><tr><td>First Response Time(Avg)</td><td>The average length of time from when the conversation actually starts in Inbox to the agent's first reply.<br />P.S.:- Current agent average response time = total first response time of the current agent / total first response conversations of the current agent)<br />- The first response is attributed to the agent who responded first.<br /></td></tr><tr><td>Resolution Time (Avg)</td><td>The length of time from when the conversation actually starts in Inbox to when the conversation is closed.</td></tr></tbody></table>

#### Agent Ranking Statistics

Sorting rule: Agents with more received conversations are ranked at the top.

![](/images/gitbook/image-399-d5f8e3ba.png)

## Inboxes

From the perspective of Inboxes, the entry is shown in the figure Analytics > [Inboxes](https://www.ycloud.com/console/#/app/inbox/analytics)

* **Data Filtering:** Select the statistical time period, defaulting to the last 7 days. Up to the last 1 year of data can be selected. Select the corresponding Inbox, defaulting to the total data of all Inboxes, click the "magnifying glass" search button to load the data.
* **Data Download:** Click the "`Download`" button

![](/images/gitbook/image-400-bf126c21.png)

#### Inbox Conversation Statistics Fields

<table><thead><tr><th width="372">Field</th><th>Description</th></tr></thead><tbody><tr><td>Conversations(Total)</td><td>The total number of received conversations.<br />P.S.: When the same conversation is closed and then reopened, it counts as 2.</td></tr><tr><td>Inbound message(Total)</td><td>The total number of received messages, not the number of conversations.</td></tr><tr><td>Outbound message(Total)</td><td>The total number of sent messages, not the number of conversations.</td></tr><tr><td>First Response Time (Average)</td><td>The average length of time from when the conversation actually starts in Inbox to the agent's first reply.</td></tr><tr><td>Resolution Time (Average)</td><td>The average length of time from when the conversation actually starts in Inbox to when the conversation is closed.</td></tr></tbody></table>

## Teams

From the perspective of Inboxes, the entry is shown in the figure Analytics > [Teams](https://www.ycloud.com/console/#/app/inbox/analytics)

**Data Filtering:** Select the statistical time period, defaulting to the last 7 days. Up to the last 1 year of data can be selected. Select the corresponding Team, defaulting to the total data of all Teams, click the "magnifying glass" search button to load the data.

* **Data Download:** Click the "`Download`" button

![](/images/gitbook/image-401-6e1d66be.png)

<table><thead><tr><th width="372">Field</th><th>Description</th></tr></thead><tbody><tr><td>Conversations(Total)</td><td>The total number of conversations belonging to the current Team during the selected time period. The number of conversations received by the agent. Based on the creation time of the conversation, the time period it belongs to is counted.</td></tr><tr><td>Online time (Total)</td><td>The total online time of all agents belonging to the current Team during the selected time period. Includes both available and away statuses.</td></tr><tr><td>First Response Time (Avg)</td><td>The average value of all agents belonging to the current Team during the selected time period. The average length of time from when the conversation actually starts in Inbox to the agent's first reply.</td></tr><tr><td>Resolution time(Avg)</td><td>The average value of all agents belonging to the current Team during the selected time period. The length of time from when the conversation actually starts in Inbox to when the conversation is closed.</td></tr></tbody></table>

**Agent Ranking Statistics**

Agent ranking statistics within the Team, sorted by the total number of conversations.
