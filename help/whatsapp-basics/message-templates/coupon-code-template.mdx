# Coupon Code Template

The coupon code template belongs to the marketing template category and is a marketing template that displays a Copy code button. After clicking, the code will be copied to the customer's clipboard.

![](/images/gitbook/image-464-30ec9ba5.png)

### Limitations <a href="#limitations" id="limitations"></a>

* The coupon code template is currently not supported on WhatsApp web client.
* The code is limited to 15 characters.
* The button text cannot be customized.
* The template is limited to a single copy code button.
