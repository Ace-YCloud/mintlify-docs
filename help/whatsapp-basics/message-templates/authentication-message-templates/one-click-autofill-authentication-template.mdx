# One-Click Autofill Authentication Template

The One-Tap Auto-Fill Authentication Template allows you to send a verification code along with a one-tap auto-fill button to users. When the user clicks the auto-fill button, WhatsApp will trigger an activity that opens your app and sends it the password or code.

![](/images/gitbook/image-444-f5b55e33.png)

The One-Tap Auto-Fill Button Authentication Template includes:

* Pre-set text: _\<VERIFICATION\_CODE> is your verification code._
* Optional security disclaimer: _For your safety, do not share this code._
* Optional expiration warning (optional): _This code will expire in \<NUM\_MINUTES> minutes._
* One-tap auto-fill button.
* Message validity: If the message is not delivered within the valid period, you will not be charged, and your customer will not see the message.

#### Limitations <a href="#limitations" id="limitations"></a>

The one-tap auto-fill button is only supported on Android. If you send an authentication template to a WhatsApp user with a non-Android device, WhatsApp will display a copy code button.

URLs, media, and emojis are not supported.
